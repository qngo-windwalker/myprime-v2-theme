Technical Requirements

BEFORE YOU ACCESS THE COURSE:

  1. Make sure you are running either Windows XP with Service Pack 3 or Windows Vista. See an IT administrator if you are not sure.
  2. Ensure you have one of the following browsers installed: Internet Explorer 7 or greater, or Firefox 3.3 or greater.
  3. Do a preliminary clearing of your cache, temporary internet files, and history:
    • For IE7, go to Tools >> Delete Browsing History >> Delete All.
    • For IE8, go to Tools >> Delete Browsing History >> UNCHECK the first box that says PreserveFavorites website data. Then, ensure all other check boxes are selected, and click Delete.
    • For Firefox, go to Tools >> Clear Recent History >> Select Everything from the Time Range to Clear drop down menu. Ensure that all check boxes are checked, and click Clear Now.
  4. Set your resolution to 1024x768.
    • On Windows XP:
      1. Open the Control Panel and select Display.
      2. On the Settings tab, under Screen resolution, drag the slider to 1024x768, and then click Apply.
      3. When prompted to apply the settings, click OK. Your screen will turn black for a moment.
      4. Once your screen resolution changes, you have 15 seconds to confirm the change. Click Yes to confirm the change, or click Noto revert to your previous setting.
    • On Windows Vista:
      1. Click the Start button.
      2. Click Control Panel.
      3. Click Appearance and Personalization.
      4. Click Personalization.
      5. Click Display Settings.
      6. Under Resolution, move the slider to the resolution 1024x768, and then click Apply.
      7. Make sure you have an audio output device of some kind, i.e. headphones or speakers.

ACCESSING THE COURSE FOR THE FIRST TIME:

  1. Visit http://www.myprime.org/acsap.
  2. Input your registration code.
  3. When prompted for a valid email address, enter the address you'd like for the program to use to send you any system notices.

PLEASE NOTE: You should only visit this page one time, in order to register for the course. If you return to this page after registering, and attempt to re-enter under the same registration number, an error message will appear.

ACCESSING THE COURSE AFTER YOU REGISTER:

  1. Visit http://www.myprime.org
  2. Input your login and password.

Brightcove Player System Requirements

End users should have one of the following browser and Flash versions installed to play Brightcove players:

  • Windows PC
  • Mac
  • Browser
Chrome 2.0 or higher Firefox 3.0.1 or higher Internet Explorer 6 SP1 Internet Explorer 7 Internet Explorer 8 Safari 3.2.2 or higher Safari 4 or higher Chrome 2.0 or higher Firefox 3.0.1 or higher Safari 3.2.1 or higher Safari 4 or higher Flash Flash Player 9, version 9.0.28 or higher Flash Player 10, version 10.0.22.87 or higher Flash Player 9, version 9.0.28 or higher Flash Player 10, version 10.0.22.87 or higher Working with the Media API through the api.brightcove.com URL over port 80 (HTTP) is rarely a problem through firewalls which have that port open. Normally, when a firewall is blocking access to the API, it can be easily determined by simply visiting http://api.brightcove.com/services/library (you should get an unknown method error: {"error": "UNKNOWN_METHOD","code":301}) from behind the firewall. If that page can't be reached via the browser, then the firewall is definitely blocking (either the URL, port 80, or a combination of the two). If the link loads correctly, it may be an issue of the AIR app using the wrong port. If this is the case, try explicitly specifying port 80 for use. Further debugging can be performed with a tool like netstat to ensure that you can reach the appropriate port on api.brightcove.com. You shouldn't need any domains other than api.brightcove.com whitelisted to access the API through the firewall. In order to use the Brightcove Studio from within your network, your browser client needs to be able to access a number of required ports and domains. If you cannot access the ports and domains below, you may experience issues using the Brightcove Studio and modules. The list of key URLs for Brightcove operation are: *.brightcove.com brightcove.vo.llnwd.net brightcove.fcod.llnwd.net *.edgefcs.net *.edgesuite.net *.akamaihd.net For the Brightcove Studio Analytics module, add: *.tubemogul.com If wildcarding is not possible, individually list the following domains (all over http port 80): admin.brightcove.com console.brightcove.com services.brightcove.com www.brightcove.com link.brightcove.com brightcove.vo.llnwd.net brightcove.fcod.llnwd.net c.brightcove.com api.brightcove.com fms.brightcove.com goku.brightcove.com reporting.brightcove.com search.brightcove.com secure.brightcove.com sadmin.brightcove.com my.brightcove.com signin.brightcove.com api.tubemogul.com load.tubemogul.com receive.inplay.tubemogul.com FMS Port 1935 (for streaming image capture only) Please understand that specific IP addresses are not available, as they may change at any time without notice. In addition, as we add new features or modify existing features, this list may change.
